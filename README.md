# Customer Segmentation using RFM Analysis

This project is part of the Data Science internship task at Syntecxhub. It aims to segment customers based on their transactional behavior using the RFM (Recency, Frequency, Monetary) model.

## 📊 Project Overview
- **Dataset Size:** 10,000 Records (Cleaned down to 8,561 rows)
- **Tech Stack:** Python, Pandas, NumPy, Seaborn, Matplotlib, Google Colab
- **Core Concepts:** Data Cleaning, Feature Engineering, RFM Scoring, Customer Segmentation

## 🚀 Key Segments Identified
- **Champions:** Best customers who buy frequently and spend the most.
- **Loyal Customers:** Regular buyers responsive to promotions.
- **New Customers:** High recency but low frequency.
- **Churn Risk:** Customers who haven't purchased in a long time; need win-back campaigns.
